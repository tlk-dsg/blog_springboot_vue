# blog_springboot_vue

## 主要目录

├── auth-center              #单点登录系统
├── blog-web             	 #博客页面
├── blog             		 #博客管理系统服务端
├── blog-admin               #博客管理系统客户端

## 技术栈

服务端：Springboot + Mybatis-Plus + Nacos
客户端：Vue.js + vuex + axios + element-ui + vue-elememnt-admin + mavon-editor
